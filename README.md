# Hi, I’m Felipe Chavarro Polania

I build governed agentic systems, evidence-bound evaluations, deterministic controllers, and reusable AI product operating models.

My work sits at the intersection of:

- AI product leadership
- trustworthy AI and agent evaluation
- security engineering
- experimental research
- governed service automation

## The question behind my work

> How can organizations use AI to accelerate work without giving free-form models uncontrolled final authority?

My operating principle is:

> **Agents propose. Humans approve. Deterministic controllers decide. AI explains.**

## Current focus

### Governed agent workbenches

Patterns for approved workspaces, versioned Skills, model routing, typed tools, human approval, and auditable artifacts.

### Evidence-bound agent evaluation

Methods for determining whether an executable action is actually supported by the current evidence and decision rule.

### Deterministic controllers

Portable rule-based systems that convert approved specifications and real evidence into reproducible decisions.

### AI product operating systems

Reusable product artifacts for turning AI ideas into controlled experiments, roadmaps, release gates, and measurable outcomes.
